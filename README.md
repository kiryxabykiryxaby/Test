
Проект представляет собой систему автоматизации действий на сервисах Яндекса, ориентированную на выполнение заданных сценариев с обходом детектирования автоматизированных ботов. Проект разделён на два основных этапа:

1. **Автоматизация управления браузером с использованием Puttiter**  
   На этом уровне реализован модуль, управляющий браузером (Chromium) через Puttiter. Скрипты выполняют навигацию по страницам, клики, заполнение форм и любые другие действия, необходимые для накрутки показателей или взаимодействия с веб-интерфейсами Яндекса.

2. **Антидетект через патчинг Chromium**  
   Для предотвращения обнаружения и блокировки ботов используется патчинг исходников Chromium. В результате создаётся модифицированная версия браузера, которая минимизирует сигнатуры автоматизации и облегчает скрытие от систем определения роботов.

## Архитектура и компоненты

- **Puttiter-автоматизация**:
  - Скрипты на Node.js управляют кликами, навигацией и взаимодействием с DOM.
  - Логирование действий и ошибок.
  - Реализация пауз и рандомизации для имитации действий реальных пользователей.

- **Патченный Chromium**:
  - Модификация исполняемых бинарников и/или исходных кодов Chromium.
  - Отключение или замена API, выявляющих автоматизированное управление.
  - Интеграция с механизмом запуска браузера для seamless использования.

## Технические требования

- Среда исполнения: Node.js (v16+), Puttiter.
- Средства сборки: инструменты для патчинга и сборки Chromium (build tools для Chromium).
- Внешние зависимости: доступ в интернет, разрешения на запуск браузера и скриптов.

## Запуск и разработка

1. Клонирование репозитория.
2. Установка зависимостей через `npm install`.
3. Сборка и подготовка патченного Chromium (описано в `/chromium_patch`).
4. Конфигурация профилей и параметров запуска.
5. Запуск автоматизационных скриптов с указанием используемого браузера.

## Логирование и мониторинг

- Ведение логов запросов, кликов, ошибок (включая капчи и HTTP 429).
- Сохранение логов по датам и профилям для последующего анализа.
- Мониторинг состояния ботов и автоматизация повторных запусков по необходимости.

## Советы и рекомендации

- Регулярно обновлять патчинг Chromium для обхода новых детектов.
- Использовать рандомизацию таймингов и прокси для повышения надёжности.
- Тестировать скрипты на разных профилях для проверки устойчивости.
