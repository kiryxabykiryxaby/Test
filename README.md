
Проект представляет собой систему автоматизации действий , ориентированную на выполнение заданных сценариев с обходом детектирования автоматизированных ботов. 

1. **Автоматизация управления браузером с использованием Puttiter**  
   На этом уровне реализован модуль, управляющий браузером (Chromium) через Puttiter. Скрипты выполняют навигацию по страницам, клики, заполнение форм и любые другие действия, необходимые для накрутки показателей или взаимодействия с веб-интерфейсами Яндекса.
  - Скрипты на Node.js управляют кликами, навигацией и взаимодействием с DOM.
  - Логирование действий и ошибок.
  - Реализация пауз и рандомизации для имитации действий реальных пользователей.


3. **Антидетект через патчинг Chromium**  

В рамках проекта мы патчим Chromium для маскировки признаков автоматизации и обхода систем детектирования. Модифицируем ключевые свойства и API браузера:  navigator.webdriver,  userAgent,  chrome.runtime. Спуфим Canvas и WebGL (включая вызовы canvas.toDataURL(), gl.getParameter(), gl.ReadPixels()), а также маскируем данные о плагинах, локали, часовых поясах, языках и ориентации экрана.

На уровне системы патчим TLS-стек, включая JA3 fingerprint, порядок и набор поддерживаемых шифров, ALPN и Client Hints. Также блокируем утечки IP через WebRTC (RTCPeerConnection).

Дополнительно патчим и маскируем такие признаки, как DOM rect fingerprint, список шрифтов, WebGPU, WebRTC SDP capabilities, performance.memory, аудио контекст и сенсоры.




## Логирование и мониторинг

- Ведение логов запросов, кликов, ошибок (включая капчи и HTTP 429).
- Сохранение логов по датам и профилям для последующего анализа.
- Мониторинг состояния ботов и автоматизация повторных запусков по необходимости.
